Project 1 — SilentBridge

Real-time Hand Gesture → Text using a webcam-based pipeline to support communication for deaf and hard-of-hearing users in online environments.

Computer Vision Webcam Inference SVM Classifier
GoalConvert hand gestures into text in real time
DataCollected gesture images via webcam
OutputPredicted gesture label → text

Visuals are adapted from the project presentation (reorganized for web readability).

Overview

Virtual meetings and online classes are increasing, but deaf and hard-of-hearing individuals can experience communication barriers when the other person does not understand sign language.

SilentBridge addresses this gap by detecting hand gestures from a webcam feed and converting them into readable text.

Method

Pipeline: data collection → preprocessing → model training → real-time inference.

Images are resized and processed into a model-friendly format. A lightweight classifier (SVM) is used for fast inference and simple deployment.

Implementation

The project follows a clear file structure: data collection scripts, training script, and an inference script that runs with a webcam.

In class, a live demo was conducted to demonstrate real-time inference. This webpage focuses on summarizing the system structure and results.

Results

The model successfully recognizes common gestures in real time. Performance can vary with lighting conditions and hand angle.

Observation: some gestures are more robust, while visually similar gestures may be occasionally misclassified.

Discussion

Real-time usability depends not only on model accuracy but also on consistent input conditions (camera quality, lighting, background).

Possible improvements include more diverse training data, stronger augmentation, and experimenting with CNN-based models for better generalization.

Conclusion

SilentBridge demonstrates a full AI development cycle and shows how gesture recognition can improve accessibility in online communication.

Future integration into platforms like Zoom or Google Meet could further expand its impact.
