Project 2 — Hourly Electricity Load Forecasting

Machine learning–based forecasting of national hourly electricity demand to support Building Energy Management Systems (BEMS).

Time Series Energy Data Regression Models
Goal Forecast hourly electricity demand
Data Weather + calendar variables
Output Predicted electricity load

Visual adapted from project presentation.

Overview

Accurate electricity load forecasting is essential for optimizing HVAC operation, energy storage scheduling, and reducing operational costs in smart buildings.

This project builds a full forecasting pipeline using historical demand and weather data.

Method

Pipeline: data preprocessing → feature engineering → model training → evaluation.

Lag features (e.g., previous-hour load) and weather variables are used to capture temporal patterns.

Implementation

Multiple regression models were implemented and compared, including baseline linear regression and ensemble models.

Performance was evaluated using MAE and RMSE metrics.

Results

The trained models successfully captured daily and seasonal demand patterns.

Lag-based features significantly improved prediction accuracy.

Discussion

Forecast accuracy depends heavily on recent load history and weather variability.

Future work may explore deep learning models such as LSTM or Transformers for longer-term forecasting.

Conclusion

This project demonstrates how machine learning can support energy-efficient building operation through accurate load forecasting.

The results highlight the importance of feature engineering in time-series prediction.
